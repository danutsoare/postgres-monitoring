About PostgreSQL Monitor

PostgreSQL Monitoring Dashboard

A comprehensive monitoring solution for PostgreSQL databases, providing real-time and historical insights into database performance, resource usage, and health.

Features
  • Real-time monitoring of database activity
  • Session resource usage tracking and identification of resource-intensive sessions
  • Wait event analysis for performance bottlenecks
  • Blocking session hierarchies visualization
  • Temporary space usage monitoring
  • Historical data collection and trend analysis
  • Support for PostgreSQL versions 10 through 15
  • Integration with PostgreSQL extensions for deeper insights
PostgreSQL Extension Support

This dashboard integrates with the following PostgreSQL extensions:

  • pg_stat_statements - For query performance analysis
  • pg_buffercache - To examine shared buffer usage
  • pg_stat_bgwriter - For background writer statistics
  • pg_wait_sampling - For detailed wait event analysis
  • pg_stat_monitor - For enhanced query monitoring
  • pgstattuple - For tuple-level statistics
  • pg_visibility - For visibility map examination
System Requirements
  • Docker Engine 20.10.0 or higher
  • PostgreSQL 10 or higher
  • Modern web browser with JavaScript enabled
